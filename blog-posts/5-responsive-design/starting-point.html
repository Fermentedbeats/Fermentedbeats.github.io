Reflections:

In this challenge we took the above page of cat pictures and veggie lists from a static page to a responsive one. Sadly, the 'test on mobile' function in Chrome's Dev Tools did not appear to work for this challenge, but otherwise we managed to reformat for a smaller screen successfully.

The biggest challenges were in stripping the formatting from the original CSS out of the directions in the @media portion of the file. The Team Treehouse tutorial set us on the right path to containing all mobile adjustments with the @media subsection of the CSS. Overall, it seems wise, that when building from scratch, the author should build for either mobile or large-screen browser first, then work towards the other end of the spectrum.

I truly enjoy tinkering with website layouts and cannot wait to try the next level of adaptive design.
