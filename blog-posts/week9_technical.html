Building a DBC Blog


Date: October 26, 2014

It's thrilling to have created my own blog, and thanks to DBC's Phase 0 deliverables, keep posting new content. I had purchased a domain and got started with Wordpress several months before Phase 0 started. While that was fun, actually crafting the HTML and CSS was rewarding on a whole new level. My next major challenge will be shoe-horning HTML & CSS of my own design to my domain without losing the handy WP plugins.

For this week's HTML challenge I sought feedback and implemented changes to my DBC website. Namely, the background and header got the boot, I fixed the scale on the brand affiliate logos, cleaned-up unused pages, and for now: abandoned the comments box. I spent some time wireframing a new design to find that I simply don't have enough material to force a site with long-copy style. I imagine this will change as I learn and geenrate a portfolio of projects to add
